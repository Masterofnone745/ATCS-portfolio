Challenge

Getaway 2

1. Locate Hidden Inputs

Right-click the page and select Inspect. Look inside the <form> tags. Often, flags are hidden in type="hidden" input fields.

2. Check the Network Tab

Open the Network tab and refresh. Look for API calls or headers that might contain the "getaway" token.

Challenge

Close Clawed

1. Console Manipulation

Sometimes "Clawed" buttons are disabled. In the Inspect Tool, select the element and remove the disabled attribute from the HTML, or use the Console to trigger the function manually.

// Try running this in your browser console:

document.querySelector('.claw-button').disabled = false;
console.log("Button Unlocked!");
