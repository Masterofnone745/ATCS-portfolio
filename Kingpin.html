KingPin: Team Selector App

KingPin is a specialized management tool designed to streamline team selection and player organization. For this project, I focused on building functional working models for key pages and refining the user experience through consistent UI and feature implementation.

Technical Contributions

1. Page creation

I created the first versions of the login, sign up, dashboard, and player view player stats pages. This involved creating and updating the html, css, and js into a efficient working model.

2. Page design

I helped design the starting layouts for the team that would later be used for the pages like the dashboard, login, and team selector pages. Mainly via comments throughout development and creating visual mockups for the team to use as reference material.

3. Bug fixing and general UI changes

I did a lot of bug fixing and UI changes throughout the development process of the website. A lot of my changes were to make the website as easy to use as possible. For example,

  • ▹ I created the working add a player and add award functions
  • ▹ I updated the matchcenter to make it connect properly with the view player stats page
  • ▹ I made the text easier to understand and more consistent throughout the site's creation.

Implementation Highlights

  • Logic Integration: Connected front-end forms to dynamic JavaScript functions for player management.
  • System Consistency: Standardized UI text and layouts to improve user flow.
  • Cross-Page Sync: Linked Matchcenter events directly to specific player statistical views.
